\subsubsection*{template$<$class T$>$\newline
class Edge$<$ T $>$}
Template class representing an edge in a graph. 

This class models an edge in a graph, holding a pointer to the destination vertex and an associated airline object. It is used within the \doxylink{class_graph}{Graph} class to represent connections between vertices.
