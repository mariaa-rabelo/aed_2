Detailed Description

template<class T>
class Edge< T >

Template class representing an edge in a graph.

This class models an edge in a graph, holding a pointer to the destination vertex and an associated airline object. It is used within the Graph class to represent connections between vertices.

Template Parameters
TThe type of content in the vertices that the edge connects.
