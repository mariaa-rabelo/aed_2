Detailed Description

template<class T>
class Vertex< T >

Represents a vertex in a graph. This class contains the information stored in a vertex, along with a list of outgoing edges (adjacency list). It also contains auxiliary fields for various graph algorithms.

Template Parameters
TThe type of the content in the vertex.
